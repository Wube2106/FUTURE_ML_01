Sales & Demand Forecasting

Predicts daily sales for retail stores using historical sales data and machine learning. The project uses time-based features, lagging, rolling statistics, and Random Forest Regression to forecast sales and provide business insights.

Features
Handles missing and incorrect data
Creates time-based features (Month, Weekday, Season)
Lag & rolling features for past behavior
Difference features to detect sudden changes
Forecasts next 30 days of sales
Visualizes actual vs predicted sales, feature importance, and forecast trends
Tools
Python, Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn (Random Forest)
Jupyter Notebook
Model Evaluation
RMSE ≈ 26 units
R² ≈ 0.94
Key features: Sales_lag_1, Rolling_mean_3, Promo_Impact
Business Insights
Sales fluctuate daily → demand is unstable
Discounts & promotions drive sales spikes
Recommendations: maintain safety stock, use promotions strategically, improve event data
